# github-js

JavaScript (jco / componentize-js) implementation of the local:github component.

> **Status: pinned to WASI 0.2.** jco@1.19 ships only `preview2-shim`; the
> `wasi:http@0.3.x` / p3-shim work lives on `bytecodealliance/jco@main` but is
> not yet released to npm. Re-evaluate when jco 1.20+ ships. The Rust and
> Python siblings have already moved to WASI 0.3 — see
> [`../github-rs/README.md`](../github-rs/README.md#per-language-quirks).

See [`../github-rs/README.md`](../github-rs/README.md) for the cross-language
overview, the shared WIT, and build/run instructions.
